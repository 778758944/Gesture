Gesture.js是什么？

Gesture.js是一个轻量级的，专为支持移动触屏设备的手势操作而设计的库。任何遵守W3C的Touch事件规范的Web浏览器环境都可以使用它。 到目前为止，所有单点触摸事件（pan，tap，press，flick）都能够在绝大多数Android设备上使用（包括浏览器和一些hybrid环境）；ios4.0及更高版本中的safari中，可以使用双指事件dualtouch。

Gesture.js使用MIT开源协议发布，在源代码中包含协议的全文。

为何要使用Gesture.js

在移动触屏设备上，想要基于W3C的标准touch事件来开发Web应用是非常困难的，比如，假定有一个列表，用户拖动它和点击其中某一列表项时都会产生一组"touchstart-touchmove-touchend"事件，在JS代码中想要区分用户的真实意图十分困难。

使用Gesture.js可以解决这些问题。我们研究了很多手机原生应用的交互，从中提取了最为常用的几个手势，并且在touch事件的基础上封装成了几个手势事件，使用这些事件就可以在Web应用中开发出接近原生应用的交互效果。

如何使用Gesture.js

首先你必须在页面上引用gesture.js文件。 

<script type="text/javascript" src="gesture.js"></script>

选择一个DOM元素，调用enableGestureEvents： 

<script type="text/javascript" >
    enableGestureEvents(document.querySelector("#myele"));
</script>

之后就可以像原生的事件一样监听Gesture事件了：

<script type="text/javascript">
    enableGestureEvents(document.querySelector("#myele"));
    document.querySelector("#myele").addEventListener("tap",function(){
        //do something
    },false);
</script>

Gesture事件皆发生在冒泡阶段，addEventListener第三个参数是无效的。

完整代码如下：

<html>
<head>
<meta charset="UTF-8" />
<title>Gesture.js示例</title>
<script type="text/javascript" src="gesture.js"></script>  
</head>
<body>
<div id="myele" style="width:50px;height:50px">点我</div>
<script type="text/javascript">

    enableGestureEvents(document.querySelector("#myele"));

    document.querySelector("#myele").addEventListener("tap",function(){
        //do something
    },false);

</script>
</body>
</html>

tap和press事件

tap是手指点触事件。tap的触发过程是手指轻点触屏并迅速离开。

手指长按触屏，将会得到press事件，当手指在触屏表面停留时间超过0.5秒时会触发press事件，触发press事件后，手指再离开触屏则会得到pressend。

这两个事件经常用于按钮形状的UI组件。

tap和press是互斥的，也就是说，触发press事件之后，手指离开触屏时将不再触发tap。

myele.addEventListener("tap",function(){
    //do something
},false);
myele.addEventListener("press",function(){
    //do something else
},false);

利用pan来改变元素的位置

pan/pan/panstart/panend 是一组响应手指在触屏上滑动的事件。与原生的touchstart/touchmove/touchend不同，只有当用户意图明显地移动手指时才会触发pan序列的事件，它和tap也是互斥的。这意味着开发者可以利用pan和tap来区分用户的真实意图。

pan事件常常用来拖拽改变元素的位置，pan事件的参数中displacementX和displacementY可以为此提供帮助。

displacementX和displacementY表示用户手指从pan开始在触屏上移动的距离，拖拽元素时，只要给元素的起始位置加上这两个值就可以实现类似拖拽的效果，请看下面代码段：

 var baseX = 0, baseY = 0;

//假设myele是某一DOM元素
myele.addEventListener("panstart",function(e){
    baseY = parseFloat(myele.style.left);
    baseX = parseFloat(myele.style.top);
},false);

myele.addEventListener("pan",function(e){
    myele.style.top = baseY + e.displacementY + "px";
    myele.style.left = baseX + e.displacementX + "px";
},false);

pan在也常用于滚动和切换场景。因为ios5.0之前的safari不支持元素内的滚动条，所以有时需要用pan和transform来模拟。

监听flick事件编写加速动画

flick事件是用手指快速在屏幕划过。

flick与pan事件是不互斥的。可以认为，flick发生之前，一定会发生pan事件。

与pan事件不同，flick是时间敏感的，只有当手指在触摸屏上停留的时间足够短，才会触发flick事件。

flick事件的参数与速度有关：

valocityX和valocityY是手指划动速度的x方向和y方向的值

duration是手指停留在触屏的时间。

flick事件的参数与时间相关，所以可以用来实现物理效果的动画。

myele.addEventListener("flick",function(e){
    //利用valocityX和valocityY来计算动画
},false);

多数模拟运动的规律都是匀加速/减速运动，所以往往需要用贝塞尔曲线模拟二次曲线，以下函数可以用[0,1]区间上的三次贝塞尔曲线拟合抛物线y=x²上[a,b]之间的一段：

 function quadratic2cubicBezier(a, b) {
    return [[(a / 3 + (a + b) / 3 - a) / (b - a), (a * a / 3 + a * b * 2 / 3 - a * a) / (b * b - a * a)],
        [(b / 3 + (a + b) / 3 - a) / (b - a), (b * b / 3 + a * b * 2 / 3 - a * a) / (b * b - a * a)]];
}

使用双指事件来做图形变换

Gesture.js提供了双指手势的支持。但是到目前为止仅ios中的safari可以提供多于一个点的touch事件，所以双指手势事件仅能用于ios。

双指手势实际包含了strech, pinch, rotate,dualpan几种，但是因为实际使用中，这些手势常常是由同一个动作触发的，也经常会需要同时使用，所以gesture.js仅仅提供了一个双指事件dualtouch。

选择性忽略某些参数的话，dualtouch可以被当做仅仅缩放、旋转或者位移的事件来使用。

为了给用户以"手指在控制缩放"的感觉，缩放和位移常常叠加使用。

在dualtouch的事件响应函数中，以下代码即可对一个元素进行缩放：

//假定myele是某一DOM元素
myele.style.webkitTransform = "translate(" + e.translate + ") "+scale(" + e.scale+ ")"

实际代码中，可能需要使用getComputedStyle来获取元素原本的transform进行叠加，所以通常可以在dualtouchstart事件中做此事。

//假定myele是某一DOM元素
var matrix;

myele.addEventListener("dualtouchstart",function(e){
    matrix = window.getComputedStyle(origin).webkitTransform;
    if(matrix == "none")
    {
        origin.style.webkitTransform = "scale(1)";
        matrix = window.getComputedStyle(origin).webkitTransform;
    }
    console.log(matrix);
},false);

myele.addEventListener("dualtouch",function(e){
    origin.style.webkitTransform = " scale("+e.scale+") "+ matrix;
},false);;

matrix则是表示完整的几何变换。。。。。。。。

若将两个触点视为一个有向线段，那么手指移动前后的有向线段可以视为一个相似变换。而scale、rotate、translate则为此相似变换的三个分量。matrix则是三种分量相叠加而得到的矩阵。

//假定myele是某myele.style.webkitTransform = " matrix("+e.matrix[0][0]+","+e.matrix[1][0]+","+e.matrix[0][1]+
    ","+e.matrix[1][1]+","+e.matrix[0][2]+","+e.matrix[1][2]+") "

函数

函数enableGestureEvents

参数：
element
任意一个DOM元素，想要在上面监听手势事件的元素。
说明：
enableGestureEvents允许用户在某一元素上使用手势事件。
var myele = document.querySelector("#myele");
enableGestureEvents(myele);
//之后就可以监听ele元素的手势事件了。

双指触摸(dualtouch)

参数：
rotate
事件发生时两手指之间连线相对于起始状态转过的角度。
scale
事件发生时两手指之间连线相对于起始状态的长度比例。
translate
事件发生时两手指之中点对于起始状态移动的距离。
matrix
两手指触点为起止的向量由起始状态做相似变换到事件发生时的状态所需的矩阵。（常用于transform属性，见示例）
touches
事件发生时两手指的触屏位置信息。
触发方式：
把两根手指放在触屏上滑动。
示例：
./samples/transform.html
./samples/zoom.html
